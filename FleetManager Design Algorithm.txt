
1. Get/input data

	1.2 If menu option selected is (A)dd, input the following:

		1.2.5 Get Length

	1.3 If menu option selected is (R)emove:

	1.4 If menu option selected is (E)xpense:

2. Perform computation
        2.1 Add expenses to boat
		2.1.1 If expenses < purchase price, expenses = expenses + amount
	2.2 Compute remaining budget

	2.3 Compute total purchase price of all boats
	2.4 Compute total expenses of all boats

3. Output data
